Swiss Ephemeris API 瑞士星历表 API

JSON struct of response 返回的JSON结构

{
    "code": 0,
    "message": "a message for this result",
    "data: {
        ...
    },
    "at": 0,
    "duration": 0
}
Field 字段 Type 类型 Description 描述
code integer 0 for success, non-zero for error.
为0表示执行成功，非0表示出错
message (optional) string error message. no this field when success.
错误信息，成功时无此字段
data null|object|any the result of this api.
API返回的结果
at integer current timestamp (millisecond).
当前时间的时间戳（毫秒）
duration float how long do this api execution(millisecond).
当前API执行时长（毫秒）
Note: The following API result refers to the value of the data field.
注意：下文的API结果的一律指上面的data字段的值

Solar 太阳相关

/solar/terms/{year}?tz=xxx Solar Terms 24节气
Example/示例： 2021 year 
{
    "year": 2021,
    "solar_terms": {
        "冬至": {
            "jd_ut": 2459219.6411574497,
            "at": "2021-12-21T23:59:08+08:00",
        },
        ...
    },
}
Field 字段 Type 类型 Description 描述
tz=Asia/Shanghai string the string of time zone.
时区
year integer the year in the url.
URL中传递的{year}
solar_terms.jd_ut float64 julian day of the solar term.(UTC)
当前节气的儒略日(世界时)
solar_terms.at string date string of the solar term with time zone.(RFC3339)
当前节气的时间(RFC3339)
